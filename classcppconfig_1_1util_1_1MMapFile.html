Detailed Description

template<typename T = char, int32_t MADVISE = POSIX_MADV_SEQUENTIAL>
class cppconfig::util::MMapFile< T, MADVISE >

Represents a memory-mapped file with optional madvise behavior.

Template Parameters
TThe type of data stored in the memory-mapped file (default is char).
MADVISEThe madvise behavior for the memory-mapped file (default is POSIX_MADV_SEQUENTIAL).
